General Info

Device name: raspberrypi
OS: Raspbian

Terminal Commands

Check Raspbian version: cat /etc/os-release

Updating Raspbian:

  • sudo apt-get update
  • sudo apt-get upgrade

Configure: sudo ifconfig

Find IP address:

    From Pi:
  • hostname -I
  • ifconfig
    From Mac:
  1. ifconfig
  2. sudo nmap -sn xxx.xxx.x.xxx/24

SSH into Pi:

  • ssh user@devicename
  • ssh xxx.xxx.x.xxx -l user

Load GUI desktop(on Pi): startx
